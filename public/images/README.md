# Images Directory

このディレクトリには、アプリケーションで使用する画像ファイルを配置します。

## 必要な画像

### CodeRabbitロゴ（横長）
- **ファイル名**: `coderabbit-logo.png`
- **形式**: PNG（背景透過推奨）
- **サイズ**: 幅200-300px程度（高さは自動調整）
- **用途**: Xシェア機能で生成する画像の右下に表示

**このファイルをアップロードしてください。**

## 配置例

```
public/images/
├── README.md（このファイル）
└── coderabbit-logo.png（アップロードが必要）
```
